Detailed Description

template<class PositionType, class Fitness, class Comparator>
class PSO::ParticleSwarmOptimization< PositionType, Fitness, Comparator >

The ParticleSwarmOptimization class is represents the PSO Algorithm. Its simulates a swarm of particles, that travel over the solution spaces to find a fit solution.

Template Parameters
PositionTypeis the type of the "position" of a particle. For example, a continuous travel would use a double or a long double.
Fitnessis the class that is used to evaluate the fitness of each particle. Such class must have a operator() that returns a PositionType, the fitness of this particle.
Comparatoris used to compare whether one particle is fitter than the other. It must have a operator(PSO_Particle A, PSO_Particle B) that returs true iff A is fitter than B.
