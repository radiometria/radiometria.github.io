Detailed Description

The TensorialPowerSeriesRouting class is a type of Power Series Routing algorithm. Differently from MatricialPowerSeriesRouting, this PSR uses the Krönecker product to produce, from a reduced set of coefficients, the large set required to calculate the power series.
