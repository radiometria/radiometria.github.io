Detailed Description

The Simulation_NSGA2_RegnPlac class runs MORP regenerator placement algorithm. This simulation uses the NSGA-2 genetic algorithm to find a suitable set of transparent and translucent nodes, in order to fit simultaneously call blocking probability, CapEx and OpEx constraints.
