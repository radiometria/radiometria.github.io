Detailed Description

The FirstNarrowestSpectrum class implements the First Narrowest Spectrum RegeneratorAssignmentAlgorithm.

The algorithm tries to save spectrum in the network by always trying to use the highest allowed spectral efficiency modulation format in the TransparentSegment.
